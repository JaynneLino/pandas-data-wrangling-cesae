# Pandas Data Wrangling & Analysis (CESAE Digital) 📊

This repository contains practical exercises and scripts developed during the **Data Analyst program at CESAE Digital**. It serves as a technical reference for data manipulation tasks using the Pandas library.

### 🧪 What's inside?
- **Data Cleaning:** Handling missing values (`NaN`), removing duplicates, and type conversion (`astype`).
- **Data Transformation:** Reading/Writing between CSV and Excel, and managing delimiters.
- **Statistical Analysis:** Calculating descriptive statistics (mean, median, mode) and data aggregation with `groupby`.
- **Filtering & Selection:** Advanced filtering using `.loc`, `.iloc`, and boolean logic.
- **Visualisation:** Basic plotting with `Matplotlib` to identify patterns and distributions.

### 🛠️ Key Snippets included
- **Group By:** Analyzing average metrics per category (e.g., salary by department).
- **Duplicate Handling:** Techniques to keep first/last occurrences or filter by specific subsets.
- **RFM Logic Foundation:** Initial steps in customer data structuring.
